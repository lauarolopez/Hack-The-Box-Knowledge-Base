# ⚡ Cheatsheets

Esta carpeta contiene mis hojas de referencia rápida (Cheatsheets).

A diferencia de las notas, aquí solo guardo comandos, sintaxis y ejemplos útiles para consultar rápidamente durante laboratorios y máquinas.

## 🎯 Objetivo

Evitar tener que buscar en Internet comandos que utilizo frecuentemente.

## 📂 Organización

Los cheatsheets estarán organizados por herramientas y protocolos.

Ejemplos:

- 🔍 Nmap
- 📂 SMB
- 🔐 SSH
- 🌐 FTP
- 📡 DNS
- 🌍 HTTP
- 🐍 Python
- 🐧 Linux
- 🛠️ Gobuster
- ⚡ FFUF
- 🧰 Burp Suite
- 🔑 Hydra
- 🖥️ Netcat

## 📖 Cada Cheatsheet incluirá

- Comandos básicos.
- Parámetros más utilizados.
- Ejemplos.
- Casos de uso.
- Tips personales.

Ejemplo:

```bash
# Detección de versiones
nmap -sV IP

# Scripts por defecto
nmap -sC IP

# Escaneo completo
nmap -A IP

# UDP
nmap -sU IP
```
