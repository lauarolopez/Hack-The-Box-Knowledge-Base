# 💻 Scripts

Esta carpeta contiene scripts desarrollados durante mi aprendizaje en ciberseguridad.

## 🎯 Objetivo

Automatizar tareas repetitivas y mejorar mi comprensión de diferentes lenguajes utilizados en pentesting.

## 📂 Lenguajes

- 🐍 Python
- 🐚 Bash
- 💠 PowerShell

## 📖 Posibles proyectos

- Port Scanner
- Banner Grabber
- Hash Identifier
- Directory Enumerator
- Reverse Shell Generator
- File Transfer Helpers

Cada script incluirá documentación sobre:

- Objetivo.
- Funcionamiento.
- Código.
- Ejemplo de uso.
- Mejoras futuras.
